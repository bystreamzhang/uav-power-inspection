---

## 项目背景

甲方为电力公司，需建设一套基于无人机机库（DJI Dock 系列）的自动化电力线路巡检系统，覆盖 500kV/220kV/110kV 输电线路、杆塔、变电站。核心诉求是替代人工巡检，实现"落地即出报告"。

## 当前前端实现（index.html）

**技术栈**：Vue 3 CDN · 纯 CSS · 单 HTML 文件 · 无构建工具
**运行方式**：浏览器直接打开，无需 npm/node

### 已实现页面

**1. 首页总览**

- 4个统计卡片：在线机场/无人机、今日任务、待处理缺陷
- 近7日巡检趋势折线图（SVG手绘）
- 机队实时状态（6台无人机 + 电量进度条）
- 最近任务列表

**2. 机场管理**

- 机场卡片：温度、电压、网络延迟、电量
- 设备接入记录表
- 绑定机场弹窗（MQTT参数：网关地址/账号/密码/组织ID/绑定码）

**3. 设备管理**

- 无人机/机库 Tab 切换
- 设备列表：型号、所属机场、状态、电量、累计飞行时长、维护到期
- 远程控制弹窗（云台/相机/喊话器/探照灯/起飞/返航）

**4. 巡检任务**

- 多条件筛选（状态/线路/类型/日期）
- 任务列表含进度条
- 新建任务弹窗（线路/航线/无人机/时间/优先级）

**5. 航线规划**

- 5种航线类型选择（正射/倾斜/航点/面状/立面）
- SVG模拟地图（航线/航点/起终点可视化）
- 航线列表（航点数/距离/时长）
- 2D/3D切换按钮（演示占位）

**6. 问题管理**

- Tab：AI智能识别 / 人工判读 / 多期对比分析
- 缺陷卡片（图标/等级/置信度/位置/状态）
- 缺陷详情弹窗 + 生成问题记录单
- 多期对比：两期巡检并排 + 变化趋势文字分析

**7. 巡检报告**

- 统计卡片（总数/本月/待审核/AI生成率）
- 报告列表（下载/预览按钮）
- 生成报告弹窗（关联任务/模板/导出格式）
- 报告详情弹窗（完整巡检信息 + 缺陷统计 + 处理建议）

**8. 数据管理**

- 4类数据统计（图像/视频/点云/缺陷标注）
- 文件列表（类型/所属任务/位置/大小/操作）

**9. 系统设置**

- AI识别参数（置信度阈值/预警间隔/目标类别/报警等级）
- 通知设置（邮箱/短信/报告生成/数据保留）

### 交互状态

所有弹窗可点击打开/关闭，导航切换正常，数据为 mock 静态数据，无真实后端。
